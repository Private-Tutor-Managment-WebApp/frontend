# Платформа для репетиторов — Frontend

Веб-приложение для частных репетиторов: управление учениками, домашними заданиями, материалами и расписанием.

## Стек технологий

- **React 18** + **TypeScript** + **Vite**
- **Tailwind CSS v4** + компоненты в стиле shadcn/ui
- **Zustand** — управление состоянием
- **React Router v6** — маршрутизация
- **date-fns** — работа с датами
- **lucide-react** — иконки

## Быстрый старт

```bash
npm install
npm run dev
```

Приложение откроется на `http://localhost:5173`.

## Функциональность

### Преподаватель

- **Главная** — статистика (задания, группы, материалы, непроверенные работы), ближайшие дедлайны, последняя активность
- **Домашние задания** — создание, редактирование, удаление заданий; привязка к группе; прикрепление материалов; оценивание и возврат работ на доработку
- **Группы** — создание и управление группами учеников; просмотр списка учеников и заданий группы
- **Материалы** — добавление и удаление учебных материалов (имитация загрузки файлов)
- **Календарь** — месячный вид с дедлайнами и регулярными занятиями; создание/редактирование/удаление расписания

### Ученик

- **Главная** — статистика, серия сдач (streak с анимацией), ближайшие задания
- **Задания** — список назначенных заданий, сдача работ, просмотр оценок и комментариев; повторная сдача при возврате на доработку
- **Материалы** — просмотр учебных материалов преподавателя
- **Календарь** — дедлайны и расписание занятий (только просмотр)

## Структура проекта

```
src/
├── types/           # TypeScript интерфейсы
├── mock/            # Моковые данные (пользователи, группы, задания, материалы, занятия)
├── store/           # Zustand сторы (auth, homework, group, material, lesson)
├── components/
│   ├── ui/          # Базовые UI-компоненты (Button, Card, Input, Dialog и др.)
│   ├── layout/      # AppLayout, Sidebar, TopBar
│   ├── auth/        # Формы входа и регистрации
│   ├── dashboard/   # StatCard, StreakCounter, RecentActivity, AssignmentCard
│   ├── homework/    # HomeworkList, HomeworkForm, SubmissionForm
│   ├── groups/      # GroupList, GroupForm
│   ├── materials/   # MaterialList, MaterialForm
│   └── calendar/    # CalendarView, LessonForm
├── pages/
│   ├── teacher/     # Страницы преподавателя
│   └── student/     # Страницы ученика
└── hooks/           # useAuth, useStreak
```

## Маршрутизация

| Маршрут                     | Описание                           |
|-----------------------------|------------------------------------|
| `/login`                    | Страница входа                     |
| `/register`                 | Страница регистрации               |
| `/teacher/dashboard`        | Главная преподавателя              |
| `/teacher/homeworks`        | Список заданий                     |
| `/teacher/homeworks/:id`    | Детали задания + оценивание        |
| `/teacher/groups`           | Список групп                       |
| `/teacher/groups/:id`       | Детали группы                      |
| `/teacher/materials`        | Управление материалами             |
| `/teacher/calendar`         | Календарь + расписание             |
| `/student/dashboard`        | Главная ученика                    |
| `/student/homeworks`        | Список заданий ученика             |
| `/student/homeworks/:id`    | Детали задания + сдача             |
| `/student/materials`        | Просмотр материалов                |
| `/student/calendar`         | Календарь + расписание             |

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`.

## Примечания

- Это фронтенд-прототип с моковыми данными, без реального бэкенда
- Авторизация через sessionStorage (сбрасывается при закрытии вкладки)
- Загрузка файлов — только имитация (сохраняется имя файла)
- Все тексты интерфейса на русском языке
